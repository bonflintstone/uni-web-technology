Question 8

Benefits of IFrames

  • IFrames circumvent the cross domain origin policy
  • Uselful to have mutliple different sources for the page
  • Security for using 3rd party content, as the iframe is basically sandboxed (javascript/ styling etc can only affect things inside the iframe)

Downsides of IFrames

  • Hard to style right
  • The content in iframes is not good for search engines to index
  • Making it easy to change the page without messing with the url
  • Can be used to update dynamically load the page without ajax
